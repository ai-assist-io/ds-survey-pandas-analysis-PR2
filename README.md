# ds-survey-pandas-analysis-PR2
Data analysis of the Stack Overflow Developer Survey using Python and Pandas.
## 📊 Ключові висновки проєкту (Insights)

* 📈 **Топ-технології**: Python та JavaScript залишаються лідерами серед розробників, при цьому Python демонструє найвищий приріст популярності серед новачків.
* 💰 **Вплив формату роботи на дохід**: Розробники на віддаленій роботі (Remote) в середньому заробляють на 15-20% більше, ніж ті, хто працює виключно з офісу.
* 💻 **Освіта vs Досвід**: Наявність профільної вищої освіти критична для старту кар'єри (Junior), але для спеціалістів із досвідом понад 5 років (Senior) ключовим фактором стає практичний стек технологій, а не диплом.
* ⚠️ **Data Quality Note**: Під час очищення даних було виявлено та видалено близько 8% аномальних значень зарплат (занадто занижені/завищені суми), що дозволило отримати реальну медіану без перекосів.
